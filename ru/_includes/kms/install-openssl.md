- Linux

  Выполните команду:
  
  ```bash
  sudo apt-get install openssl
  ```

- Windows

  Воспользуйтесь пакетным менеджером [Chocolatey](https://chocolatey.org/install). От имени администратора выполните команду в терминале PowerShell:

  ```powershell
  choco install openssl
  ```
